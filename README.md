Overview:
IronPulse Gym Management System is a browser-based web application for managing gym membership, BMI calculation, workout schedules, 
and login functionality for admins, members, and users.

Features:
Responsive Design: Works on desktop and mobile devices.
BMI Calculator: Users can calculate BMI by entering weight and height.
Membership Registration: Users can register for Basic, Pro, or Elite plans.
Login System: Supports three roles:
Admin: Access admin dashboard.
Member: Access member dashboard.
User: Access user dashboard.
Members can also log in as users.
Workout Schedule: Displays weekly workout plan.
Gym Gallery: Showcases gym images.
